Class CmdHandler

Command Handler with the following key features:

  • Re/load and re/create configuration file for commands (config/commands.json).
  • Return commandList, which has been loaded before.
  • Re/load command instances from command directory.

Static getCmdList

  • getCmdList(): any
    • Defined in src/core/CmdHandler.ts:88

    Returns current loaded list of commands containing an instance of class and filePath.
    Disabled commands won't be listed here.

    {
    instanceCmd: {
        fn : [instanceCmd],
        path : "filePath"
    },
    HelpCmd : {
        fn: [HelpCmd],
        path: "filePath"
    },
    ...
}

Static loadCmdList

  • loadCmdList(): void
    • Defined in src/core/CmdHandler.ts:99

    Re/loads all existing commands in command directory as well as their modules.
    All files listed in commands directory will be recompiled by TSCompiler. Already cached files by require will be removed. Afterwards the FileLoader will reload all *.cmd.js instances as well as their imported modules. Configuration file will take impact on loaded commands.

    Returns void

Static loadConfig

  • loadConfig(): any
    • Defined in src/core/CmdHandler.ts:40

    Loads current command configuration file from config directory and returns it. Configuration file contains key value pairs, where a key is an instance and the value defines whether the command is active.

    HelpCmd : true,
    PrideCmd : true,
    ...
}
